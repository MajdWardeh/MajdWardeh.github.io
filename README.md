# MajdWardeh.github.io

Modern, single‑page portfolio for Majd Wardeh built as a zero‑build React app and served via GitHub Pages.

Tech
- React 18 via CDN, rendered with ReactDOM (no bundler)
- Babel Standalone for inline JSX in `index.html`
- Pure CSS (light theme) with CSS variables

Local Development
- Open `index.html` directly in a browser. No build step.
- Edit content inside the React components in `index.html`.
- Replace `cv_photo_1.jpg` with your preferred image if needed.

Deployment
- Push to `main` on this repo (`MajdWardeh.github.io`). GitHub Pages serves `index.html` automatically.
